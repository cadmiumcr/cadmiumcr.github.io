Cadmium

Natural Language Processing library for Crystal

Get Started

Installation

First add cadmium to your shard.yml

{% highlight yaml %} dependencies: cadmium: github: cadmiumcr/cadmium {% endhighlight %}

Then require it in your project

{% highlight ruby %} require "cadmium" {% endhighlight %}
Check the guide for usage instructions.

Contents

Cadmium currently contains:

  • Classifiers
  • String Distance
  • GloVe
  • Graphs
  • Inflectors
  • Language Detection
  • Lemmatizers
  • N-Grams
  • Phonetics
  • POS Tagger
  • Readability
  • Sentiment Analysis
  • Stemmers
  • Summarizer
  • Tf-Idf
  • Tokenizers
  • Transliterator
  • Tries
  • Wordnet
